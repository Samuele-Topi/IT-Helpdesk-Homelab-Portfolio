# 🛠️ IT Helpdesk & Systems Administration Portfolio

Welcome to my technical portfolio. This repository documents a series of practical, hands-on laboratories designed to simulate enterprise IT support and systems administration environments. 

The primary goal of this project is to demonstrate my ability to deploy, configure, secure, and troubleshoot enterprise systems (Windows Server, Active Directory, Group Policies) and helpdesk operations (osTicket ticketing workflows, Linux server administration).

---

## 🏗️ Portfolio Structure

This portfolio is divided into two primary labs, mapping to core competencies in IT Support Tier 1/2:

### 1. [Active Directory & Group Policies Homelab](active-directory/README.md)
* **Status:** Completed
* **Core Technologies:** Windows Server 2022, Windows 10/11 Enterprise, VirtualBox, Active Directory DS, Group Policy (GPO), DNS.
* **Key Scenarios:** Domain join, user provisioning (RBAC), GPO implementation (drive mapping, control panel restrictions), and network troubleshooting.
* **Write-up:** [Read the AD Lab documentation](active-directory/README.md) | [Leggi in Italiano](active-directory/README_IT.md)

### 2. [osTicket Ticketing System Homelab](osticket/README.md)
* **Status:** Completed (AD integration completed)
* **Core Technologies:** Ubuntu Server 22.04 LTS, Apache2, MySQL, PHP v8.2, osTicket.
* **Key Scenarios:** LAMP stack deployment, osTicket installation & hardening, departments/SLA setup, ITIL-compliant ticket workflow simulation (Giulia Bianchi's network connectivity issue), and advanced AD LDAP integration.
* **Write-up:** [Read the osTicket Lab documentation](osticket/README.md) | [Leggi in Italiano](osticket/README_IT.md)
